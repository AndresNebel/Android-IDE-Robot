El lenguaje se configura desde el archivo yatay.js el la siguiente sentencia:
document.write('<script type="text/javascript" src="generated/' 
//+ BlocklyApps.LANG + 
+ 'es' +
'.js"></script>\n');

Se forzo a seleccionar español (cambiar para dar la opcion), puesto que BlocklyAps.LANG obtiene ingles como lenguaje predeterminado.
